Welcome to the MBARI Media Managment (M3) Server

This server is running NGINX. It is also acting as a proxy for the M3 REST services. The source code for the NGINX docker build is located at m3-quickstart/docker/nginx

M3 REST Endpoints

  • Configuration (raziel):
    • Endpoint: http://[servername]/config
    • Test: Show health of all M3 services
  • Annotations (annosaurus):
    • Endpoint: http://[servername]/anno
    • Test: Find a few imaged moments
  • Knowledgebase (vars-kb-server):
    • Endpoint: http://[servername]/kb
    • Test: Show phylogeny branch down to the genus Nanomia
  • User Accounts (vars-user-server):
    • Endpoint: http://[servername]/accounts
    • Test: Show all users
  • Video Asset manager (vampire-squid):
    • Endpoint: http://[servername]/vam
    • Test: Show all deployment IDs (aka video sequence names)
  • Simple Data Access (Charybdis):
    • Endpoint: http://[servername]/references
    • Test: Find all annotations and media for "Pandalus"
    • Test: Show all annotations and media for deployment "Ventana 1234"
    • Test: Show all annotations for a video file named "V4100_20180226T215822Z_prores.mov"
